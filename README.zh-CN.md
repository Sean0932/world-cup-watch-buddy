世界杯观赛搭子是一个为 REDSkill 世界杯活动设计的轻量 Agent Skill。

它不是世界杯百科，也不是复杂比分预测工具，而是一个更适合普通用户玩起来、截图、发小红书的互动生成器：

```text
一句话触发 -> 问 5 个问题 -> 生成球迷人格 -> 给观赛建议 -> 生成小红书发布素材
```

[English](./README.en.md)

---

## 适合谁

- 想参与 `#REDSkill`、`#对世界杯动手了` 活动，但不想做复杂工具的人。
- 看世界杯主要是气氛、社交、球星、名场面，不想硬装战术大师的人。
- 想知道今晚到底值不值得熬夜的人。
- 想发一篇有趣小红书笔记，但不知道标题、封面和正文怎么写的人。
- 想测试其他 agent 是否能正确使用一个 GitHub Skill 的人。

## 一句话效果

用户只要说：

```text
测测我是世界杯哪种球迷，顺便帮我安排今晚看球。
```

Skill 会先问 5 个问题：

```text
1. 你看球更像哪种？
   A 真球迷 / B 气氛组 / C 大赛限定 / D 球星颜粉

2. 最晚能接受几点睡？
   A 22 点前 / B 23 点-0 点 / C 凌晨 1-2 点 / D 能熬到 3 点以后 / E 通常无压力

3. 有支持的球队或球星吗？
   自由回答，例如：阿根廷、梅西、姆巴佩、没有固定支持

4. 看球主要为了什么？
   A 懂球 / B 社交 / C 发小红书 / D 纯快乐

5. 更怕什么？
   A 错过名场面 / B 第二天废掉 / C 朋友问你看法你完全接不住

你可以直接回复：1B, 2D, 阿根廷, 4B, 5A
```

然后输出一张结果卡：

```text
你的球迷人格：朋友圈气氛组

发疯指数：92%
今晚熬不熬：可以熬，但只熬有社交价值和名场面概率的比赛。
观赛搭子建议：阿根廷比赛优先看全场；非阿根廷比赛只看强强对话、淘汰赛、补时/点球这种高讨论度场面。
发圈文案：球我可以看不懂，但阿根廷的气氛我必须到位。
小红书标题：我让 Skill 测了下世界杯人格，结果我是朋友圈气氛组
封面大字：测出我的球迷人格了
话题：#REDSkill #对世界杯动手了 #世界杯聊个球
```

## 完整效果示例

### 示例 1：气氛组

```text
搭子 ❯ 来，先测测你是哪种世界杯球迷。你可以直接回复：
       1B, 2D, 阿根廷, 4B, 5A

用户 ❯ 1B, 2D, 阿根廷, 4B, 5A

搭子 ❯ 你的球迷人格：朋友圈气氛组

       发疯指数：92%

       今晚熬不熬：
       可以熬，但只熬有社交价值和名场面概率的比赛。

       观赛搭子建议：
       你是气氛组，但不是随便凑热闹型。
       你支持阿根廷，又怕错过名场面，所以最适合的策略是：
       阿根廷比赛优先看全场；非阿根廷比赛只看强强对话、
       淘汰赛、补时/点球这种高讨论度场面。

       发圈文案：
       球我可以看不懂，但阿根廷的气氛我必须到位。
```

### 示例 2：球星颜值应援官

```text
用户 ❯ 颜粉；年龄大了，估计能熬到 03 点；
       卡颜值；社交；朋友间交流插不上话

搭子 ❯ 你的球迷人格：球星颜值应援官

       发疯指数：82%

       今晚熬不熬：
       可以熬到 03 点，但只建议熬“颜值主场 + 朋友会聊”的比赛。

       观赛搭子建议：
       你不是硬核战术型球迷，你的核心需求是看得开心、
       聊得上话，最好还有能截图/发圈的球星瞬间。

       朋友间交流话术：
       我不装懂战术，我主要看球员状态和场上气质，但这场节奏确实挺紧。

       发圈文案：
       今晚不是看球，是看我的世界杯门面营业。
```

### 示例 3：不想熬夜

```text
用户 ❯ 气氛组；1 点前睡；没有固定支持球队；发小红书；怕第二天废掉

搭子 ❯ 你的球迷人格：第二天保命派

       发疯指数：70%

       今晚熬不熬：
       不建议硬熬。你更适合明早看集锦和赛后名场面。

       观赛搭子建议：
       如果不是你的主队、淘汰赛、决赛或全网刷屏焦点战，
       就不要把普通比赛熬成生死战。

       发圈文案：
       爱足球，也爱明天还能正常上班的自己。
```

## 它能做什么

| 能力 | 输出 |
|---|---|
| 球迷人格测试 | 8 种人格之一：朋友圈气氛组、球星颜值应援官、补时玄学大师、第二天保命派等 |
| 发疯指数 | 0-100% 的轻量娱乐评分 |
| 观赛建议 | 今晚熬不熬、看全场/半场/集锦、哪类比赛值得看 |
| 社交话术 | 给小白、颜粉、气氛组准备能接上朋友聊天的话 |
| 小红书素材 | 标题、封面大字、正文草稿、活动话题 |
| 实时数据兜底 | 需要赛程/比分/晋级时先查询或要求用户提供，不编造 |

## 为什么不是世界杯百科

已经有资料型世界杯 Skill 更适合做新赛制、球队、球星、战术百科。

这个 Skill 的定位是：

```text
用最短路径让普通用户玩起来，并产出能发出去的结果。
```

所以它刻意不做：

- 大而全世界杯资料库。
- 实时比分追踪器。
- 赌博/赔率/押注建议。
- 复杂夺冠概率模型。

它只做一件事：

```text
帮用户知道自己是哪种球迷，并决定今晚怎么舒服地看球。
```

## 安装

这个仓库本身就是一个 Skill 目录。根目录包含：

```text
SKILL.md
references/
scripts/
```

### 方式一：让 Agent 帮你安装

打开你正在使用的 Agent，比如 Codex、Claude Code、OpenClaw、小龙虾等，直接说：

```text
帮我安装这个 skill：https://github.com/Sean0932/world-cup-watch-buddy
```

安装完成后，重启 Agent。

### 方式二：使用通用 skills 安装器

```bash
npx skills add Sean0932/world-cup-watch-buddy
```

如果你的安装器支持指定 runtime，可以按自己的工具补充参数，例如 `-a codex`、`-a claude-code`、`-a openclaw`。

### 方式三：手动安装

| Runtime | 推荐安装路径 |
|---|---|
| Codex | `~/.codex/skills/world-cup-watch-buddy/` |
| Claude Code | `~/.claude/skills/world-cup-watch-buddy/` |
| OpenClaw / 小龙虾 | `~/.openclaw/workspace/skills/world-cup-watch-buddy/` |
| 其他 Agent | 放到对应 runtime 的 `skills/` 目录 |

以 Codex 为例：

```bash
mkdir -p ~/.codex/skills
git clone https://github.com/Sean0932/world-cup-watch-buddy.git \
  ~/.codex/skills/world-cup-watch-buddy
```

装好后重启 Agent，然后说：

```text
请使用 世界杯观赛搭子 这个 skill，测测我是世界杯哪种球迷。
```

## 快速试用 Prompt

安装后可以直接复制下面任意一句：

```text
请使用世界杯观赛搭子，测测我是世界杯哪种球迷，顺便帮我安排今晚看球。
```

```text
我是气氛组，可以熬夜，支持阿根廷，主要为了社交，怕错过名场面。帮我生成结果卡和小红书文案。
```

```text
我是球星颜粉，最多熬到 3 点，朋友聊球我经常插不上话。帮我测人格，并给几句朋友间交流话术。
```

```text
我不想熬夜，但想参与世界杯话题。帮我测测是哪种球迷，再给我一个不熬夜的观赛方案。
```

如果 Agent 已经列出 5 个问题，可以用编号快速回答：

```text
1B, 2D, 阿根廷, 4B, 5A
```

也可以混合自由文本：

```text
1D, 2C, 姆巴佩, 4B, 5C
```

## 实时赛程规则

这个 Skill 不内置实时赛程，也不会靠模型记忆编比赛。

遇到这些问题：

```text
今晚看哪场？
今天有什么比赛？
最新比分是多少？
谁晋级了？
谁首发？
谁伤停？
```

Agent 必须先查官方或可信来源，例如：

- FIFA 官方赛程页。
- 用户粘贴的赛程。
- 可信体育平台。

如果无法确认，必须降级：

```text
我现在不能确认今晚官方赛程。你把今晚赛程发我，我帮你判断哪场最值得看。
```

## 本地测试

可以用脚本快速测试人格分类：

```bash
python3 scripts/score_persona.py \
  "气氛组 可以熬夜 阿根廷 兴趣 错过名场面" \
  --json
```

示例输出：

```json
{
  "persona": "朋友圈气氛组",
  "persona_key": "social",
  "madness_index": 92,
  "summary": "看球不一定最懂，但气氛一定到位。"
}
```

再测一个保命派：

```bash
python3 scripts/score_persona.py \
  "气氛组 1点前睡 没有支持球队 为了发小红书 怕第二天废掉" \
  --json
```

## 诚实边界

- 不提供赌博建议。
- 不输出“必赢”“稳赚”“一定进球”等确定性判断。
- 不编造赛程、比分、首发、伤停和晋级信息。
- 不鼓励用户无差别熬夜。
- 用户表达睡眠顾虑时，优先给回放、集锦、只看关键时段等方案。
