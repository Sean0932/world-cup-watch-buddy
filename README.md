# 世界杯观赛搭子.skill

> 测测你是哪种世界杯球迷，顺手安排今晚看球。

世界杯观赛搭子是一个轻量对话型 Agent Skill：先用 5 个问题测出用户的球迷人格，再结合用户作息、支持球队和最新赛程，生成观赛建议、小红书标题、封面文案和发圈文案。
