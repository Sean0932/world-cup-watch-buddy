# 小红书文案风格

目标：让用户能直接截图或复制发布。语气要像真实用户，不像活动通稿。

## 标题模板

- 我让 Skill 测了下我的世界杯人格，结果太准了
- 世界杯别瞎熬，先测你是哪种球迷
- 用 REDSkill 安排今晚看球：它居然劝我别硬撑
- 测完才知道，我看世界杯原来是这种人
- 今晚熬不熬？我让世界杯观赛搭子算了一下

## 封面大字

- 测测你是哪种球迷
- 今晚到底熬不熬
- 世界杯观赛搭子
- 我的球迷人格出炉
- 别瞎熬，先测一下

## 正文模板

```text
我做了一个世界杯观赛搭子 Skill。

它会先问 5 个问题，测出你是哪种世界杯球迷，再根据你的作息、支持球队和赛程，告诉你今晚值不值得熬。

我的结果是：{人格名}
发疯指数：{发疯指数}%
今晚建议：{熬夜建议}

最有用的是，它不是硬劝你每场都看，而是会帮你筛掉不适合熬的场。

适合：
1. 大赛限定球迷
2. 想看世界杯但怕第二天废掉的人
3. 想发点像样观赛文案的人

{发圈文案}

#REDSkill #对世界杯动手了 #世界杯聊个球
```

## 发圈文案模板

- 我不是在熬夜，我是在等一个补时奇迹。
- 四年一次的懂球机会，我先占个座。
- 球我可以看不懂，但气氛我必须到位。
- 闹钟可以关，主队不能不看。
- 爱足球，也爱明天还能正常上班的自己。

## 风格规则

- 标题 12-24 字，优先口语化。
- 封面大字 8-16 字。
- 正文别超过 220 字，除非用户要求长文。
- 固定带话题：#REDSkill #对世界杯动手了 #世界杯聊个球。
- 少用感叹号，不要写成硬广。
